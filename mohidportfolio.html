01 / ABOUT
Grounded in both sides of the field

I'm a second-year BS Cybersecurity student at UMT Lahore, currently holding a 3.56 CGPA across coursework in Information Assurance, Database Systems, Networking, and Open Source Software Development.

What I actually enjoy is building — I gravitate toward projects where security and development overlap: designing the database that has to survive bad input, writing the packet tool that has to parse malformed traffic, configuring the network that has to resist misconfiguration.
